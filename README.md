## 🚀 About Dart

Dart is a modern, object-oriented programming language developed by Google. It’s optimized for building fast, scalable, and cross-platform apps — especially when paired with Flutter for mobile, web, and desktop development.

### ✨ Key Features of Dart
- **Fast and compiled**: Dart compiles to native code and JavaScript for high performance.
- **Productive tooling**: Includes tools like `dartfmt`, `dartanalyzer`, and `pub`.
- **Null safety**: Eliminates null reference errors at compile time.
- **Async support**: Built-in support for `Future`, `Stream`, and `async/await`.
- **Versatile**: Ideal for mobile, web, server-side, and CLI development.
